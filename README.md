# Golf App (Full Stack SaaS)

## Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express
* Database: MongoDB
* Auth: JWT

---

## Features

* User Register & Login
* JWT Authentication
* Subscription Activation
* Add Scores
* Run Draw & Winners
* Dashboard with user data
